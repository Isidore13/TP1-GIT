Question 7)

- Le git show permet d'afficher les infos sur un commit en particulier, dont on aura spécifié le hash au préalable.

- Le git status permet de voir si des fichiers/modifications sont présentes dans le Working directory ou la Staging area.

- Le git diff permet de voir les différence entre deux versions d'un fichier, et donc de voir les ajouts ou suppressions sur celui-ci.

- Le git log permet d'afficher l'ensemble des commits présents dans une branche avec l'auteur de celui-ci et la date de réalisation.
